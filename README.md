# Tanden Open Handbook

株式会社Tandenのオープンハンドブックです。会社の文化・制度・働き方などを公開しています。

## 技術スタック

- [Next.js](https://nextjs.org/)
- [Fumadocs](https://fumadocs.dev/)

## 開発

```bash
pnpm dev
```

http://localhost:3000 でプレビューできます。

## ドキュメントの追加・編集

`content/docs/` 配下に `.mdx` ファイルを追加・編集してください。

## デプロイ

[Vercel](https://vercel.com) でホスティング。`main` ブランチへのpushで自動デプロイされます。

https://tanden-docs.vercel.app
