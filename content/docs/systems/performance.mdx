title: パフォーマンス改善
description: 本人だけの問題にしない。周囲も同じだけ責任を持つ改善フロー
---

パフォーマンスの課題は、本人だけの問題ではありません。「コンテキストを受け取る」はバリューですが、コンテキストを渡す側の責任も同じだけ重いです。鍵穴がズレているのに鍵のせいにするのは、成熟した振る舞いではありません。

## 前提：なぜ「周囲の責任」なのか

パフォーマンスが出ていない状態には、いくつかの原因があります。

| 原因 | 責任の所在 |
|---|---|
| 魂のカタチとロールが噛み合っていない | アサインした側・本人の双方 |
| コンテキストが渡されていない | コンテキストを渡す側 |
| 期待値が言語化されていない | 期待している側 |
| スキルが足りていない | 本人＋学びの機会を提供する周囲 |
| 本人の状態が悪い（体調・モチベーション） | 気づけなかった周囲＋開示しなかった本人 |

「あの人パフォーマンス低いよね」と思ったとき、自分がその人に対して何をしたか（していないか）を最初に振り返ります。

## 4段階の改善フロー

### ステップ1：対話する

マイコーチ、または関係の深い同僚との対話から始めます。

- 本人が気づいていない場合もあります。[Okuri（ピアフィードバック）](/docs/tools/okuri)で率直に伝えます
- **周囲がやること**：「何に困っているか」「コンテキストは足りているか」「魂のカタチと合っているか」を一緒に見に行きます
- **本人がやること**：自分の状態を開示します。力が湧いていないなら、なぜかを言語化します

ほとんどのケースはここで解決します。対話する前に「ダメだ」と判断するのはダサいです。

### ステップ2：改善テイアンを出す

対話を経ても構造的な問題が見える場合、[テイアン](/docs/systems/teian)を出します。本人からでも周囲からでも構いません。

**テイアンの内容例：**
- 「この業務のコンテキスト共有を増やしたい」（周囲の改善）
- 「このスキルを伸ばすために学習プールからファンディングしたい」（本人の改善）
- 「期待値を再設定したい」（双方の改善）
- 「1on1の頻度を上げたい」（関係性の改善）

改善テイアンは「あなたを直す」ではなく、「この状況を直す」という視点で書きます。

### ステップ3：ロール変更テイアンを出す

改善に取り組んだ上で、鍵穴自体が合っていないと判断した場合、ロール変更を検討します。

- [Noroshi（ジョブボード）](/docs/tools/noroshi)で別のプロジェクトやロールを探します
- 本人が手を挙げることも、周囲が「こっちの方が合うのでは」と提案することもあります
- [離脱のレベル1（鍵穴を変える）](/docs/hiring/leaving)と同じ考え方です

ロール変更はネガティブではありません。鍵穴が変わっただけです。

### ステップ4：減給テイアンを出す

ステップ1〜3を経ても改善しない場合の最終手段です。

- テイアンである以上、コンテキストと材料を揃える責任は提案者にあります
- 「ステップ1〜3で何をしたか」を記録に含めます。いきなりここに飛ぶことはできません
- 感情ではなく、[ステージ](/docs/finance/salary)の期待値に基づいて判断します
- 最終判断は代表が行います

### ステップ4の先

Tandenは会社からの解雇を前提としていません。減給テイアンが通った後も改善しない場合、減給された状態のまま継続します。本人が「この環境では力が湧かない」と判断すれば、[離脱](/docs/hiring/leaving)を選びます。やめ時は本人が判断するのが原則です。

## 周囲の責任チェックリスト

「パフォーマンスが低い」と感じたとき、まず自分に問いかけます。

- その人にコンテキストを十分渡しているか？
- 期待値を言語化して共有しているか？
- その人の[魂のカタチ](/docs/culture/values/soul-shape)を見に行ったか？
- [Meibo](/docs/tools/meibo)や[Tsuzuri](/docs/tools/tsudzuri)でその人の状態を把握しているか？
- フィードバックを直接伝えたか？陰で言っていないか？
- その人の力が湧く仕事を一緒に探したか？

全部にYesと言えないなら、まず自分がやるべきことがあります。

## イケてる／ダサい

**イケてる：** 「自分が何をできていないか」から振り返る。対話と改善テイアンを経た上で、必要なら厳しい判断もする。

**ダサい：** 陰で「あの人ダメだよね」と言う。対話もせずにいきなり減給テイアンを出す。本人だけの問題にする。
