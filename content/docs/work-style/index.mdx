description: Tandenの働き方——リモート・フルフレックス・非同期中心

Tandenの働き方の基本は「場所・時間・やり方を自分でデザインする」です。フルリモート・フルフレックス・非同期を前提に、必要なときだけ同期します。制度や仕組みの面は[制度](/docs/systems)を、お金に関することは[ファイナンス](/docs/finance)を参照してください。
