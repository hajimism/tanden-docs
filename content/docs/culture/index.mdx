---

Tandenの行動原則は6つの層で構成されています。上から順に「なぜ存在するか」から「どんな空気か」まで具体化していきます。

| 層 | 問い | 内容 |
|---|---|---|
| **ミッション** | なぜ存在するか | 人の力が湧いてくる社会をつくる |
| **ビジョン** | どこに向かうか | 社会にパワースポットを増やす |
| **バリュー** | 何を大切にするか | コンテキストを受け取る・魂のカタチを活かす・成熟に向かう |
| **スタンス** | 自分を何者と捉えるか | 誰もがコンテキストデザイナーである |
| **プラクティス** | だからどう振る舞うか | 10の行動指針 |
| **ムード** | どんな空気か | 凛として、あたたかい。軽やかで、深い。熱くて、涼しい。遊ぶように、本気で。居心地がいいのに、甘くない。 |
