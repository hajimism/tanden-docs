title: 給与
description: バンド制をベースに、本人がテイアンする給与制度

Tandenには人事評価がありません。従来の人事評価業務を**分散化**し、給与は本人がテイアンで提案する仕組みです。「自分の価値は自分で言語化する」という自律の考え方が根底にあります。

## バンド制

給与はバンド（等級）ごとのレンジで設定します。各バンドはTandenのバリューに紐づいた振る舞いの期待値で定義されます。

| バンド | 期待される振る舞い |
|---|---|
| **B1：受け取る** | コンテキストを受け取り、自分の仕事に活かせている。魂のカタチを把握し始めている。プラクティスを実践している |
| **B2：活かす** | 魂のカタチを発揮し、周囲に価値を届けている。テイアンを自発的に出している。他者の学びを促進している |
| **B3：デザインする** | 他者やチームのコンテキストをデザインしている。新しい仕組みや改善を主導している。組織の成熟を牽引している |
| **B4：牽引する** | 組織全体の方向性に影響を与えている。事業やプロダクトの意思決定を担っている。社外にもパワースポットを広げている |

- バンドは固定的な「ランク」ではなく、現時点の振る舞いに対する目安です
- 複数バンドにまたがる振る舞いをしている人も普通にいます。主軸がどこかで判断します
- バンド間のレンジは重複します。B2上位とB3下位は同等の金額帯もありえます
- 各バンドの年収レンジは市場相場をベースに設定し、全社公開します

## 給与テイアンのフロー

1. 本人が給与テイアンを出します。タイミングも頻度も自由です
2. 下記テンプレートに沿って材料を徹底的に揃えます
3. 社長以外の周囲がレビュー・フィードバックします
4. 代表が昇給幅を決定し、コミットします

### 代表の判断基準

- 周囲のレビューは参考にしますが、近しい人のレビューは高く出がちです
- そのため**市場相場**を重要な判断材料とします
- 昇給幅が0の場合もあります
- 経営の持続性を守るセーフティネットとしての役割です

## 減給テイアン

パフォーマンスや振る舞いに課題がある場合、周囲から**減給テイアン**を出すこともできます。

- テイアンである以上、コンテキストと材料を揃える責任は提案者にあります
- 感情ではなく、バリューの体現度とバンドの期待値に基づいて判断します
- 減給テイアンを受けた本人は、レビューを踏まえて改善に取り組むか、ロール変更を検討します
- 最終判断は昇給と同じく代表が行います

## イケてる／ダサい

**イケてる：** 自分の価値を材料揃えて堂々と言語化する。市場相場も調べた上で提案する。

**ダサい：** 材料なしに「もっと欲しい」と言う。自分の振る舞いを棚に上げて金額だけ主張する。
