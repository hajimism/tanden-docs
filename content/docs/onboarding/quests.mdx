[Meibo](/docs/tools/meibo)のプロフィールに発行されるクエスト一覧です。達成すると[Meguriポイント](/docs/finance/meguri)を獲得できます。

クエストは「やらされるタスクリスト」ではなく、**自分のペースで選んで進めるもの**です。順番も自由。気になるものから手をつけてください。

## 制度の安心

「わからないことがない。わからなくてもAIや同僚に聞いてすぐ解決できる自信がある」状態を目指します。

| クエスト | やること |
|---|---|
| ハンドブック完走 | OpenHandbookを通読し、気になった点を[ojtチャンネル](/docs/work-style/ojt-channel)に投稿 |
| シオリエに聞いてみる | 制度やルールについて[シオリエ](/docs/tools/shiorie)に質問してみる |
| 先輩のテイアンを3本読む | 過去の[テイアン](/docs/systems/teian)を3本読んで、空気感を掴む |
| はじめてのテイアン | 何でもいいのでテイアンを1本出す。しょぼくていい、的外れでもいい |
| ハンドブック改善テイアン | ハンドブックの誤字・わかりにくい説明・足りない情報を見つけてテイアンを出す |
| オンボクエスト追加テイアン | 「こういうクエストがあったらよかった」をテイアンで提案する |
| テイアンにレビューを贈る | 他の人のテイアンにレビューコメントを書く |
| 経費を使ってみる | [事例集](/docs/finance/expenses)を読んで経費を1回使う |
| Noroshiを覗く | [Noroshi（ジョブボード）](/docs/tools/noroshi)を開いて、今どんなプロジェクトが動いているか見てみる |
| Karuteを読む | [Karute](/docs/tools/karute)で自分のレーベルの指標を眺めてみる |
| Principlesを書く | 自分の判断原則（[Principles](/docs/culture/stance)）を3つ以上書き出してojtチャンネルに投稿 |
| イケてる/ダサいを書く | 自分の仕事における[イケてる/ダサい](/docs/culture/practices)を言語化してojtチャンネルに投稿 |

## 自己の確信

「[魂のカタチ](/docs/culture/values/soul-shape)を活かせる自信がある」状態を目指します。

| クエスト | やること |
|---|---|
| 魂のカタチ診断 | 基礎診断を受けて結果を[Meibo](/docs/tools/meibo)に登録 |
| プロフィール充実 | スキル・自己紹介・写真をMeiboに登録 |
| はじめてのTsuzuri | [Tsuzuri](/docs/tools/tsudzuri)（日報）を1回書く |
| 1週間のTsuzuri | Tsuzuriを5日連続で書く |
| 1ヶ月のTsuzuri | Tsuzuriを20日書く（連続でなくてOK） |
| 「ついつい」を言語化 | 自分の「ついついやってしまうこと」をojtチャンネルに書く |
| ステッカーを5回もらう | [Tsuzuri](/docs/tools/tsudzuri)やojtチャンネルの投稿にステッカーを5回もらう |
| 自分の[ムード](/docs/culture/atmosphere)を書く | Tandenで働いてみて自分が感じている空気感をojtチャンネルに書く |
| 二項対立を超えてみる | 仕事の中で「AかBか」ではなく[両方を活かす判断](/docs/culture/values/maturity)をした経験をojtチャンネルに書く |

## 関係性の実感

「価値観の合う人と出会えている。人生を楽しんでいる人たちと一緒に働いている感覚がある」状態を目指します。

| クエスト | やること |
|---|---|
| マイコーチ設定 | [マイコーチ](/docs/systems/my-coach)とのConnectionを[Meibo](/docs/tools/meibo)に登録 |
| マイコーチと初回1on1 | マイコーチとの最初の1on1を実施 |
| 1on1を3人と | 紹介された社員と1on1を実施 |
| 1on1を5人と | 紹介以外も含め、合計5人と1on1を実施 |
| はじめてのOkuri | 誰かに[Okuri](/docs/tools/okuri)（フィードバック）を送る |
| Okuriを3人に送る | 3人の異なる相手にOkuriを送る |
| マチネに参加 | [金曜のマチネ](/docs/work-style/matinee)に1回参加する |
| マチネで発表する | マチネでLTや共有を1回やってみる |
| 他人のMeiboに書く | 一緒に働いた人のプロフィールに強みを書き加える |
| 他レーベルの人と話す | 自分の[レーベル](/docs/org-design/label)以外の人と1on1やランチをする |
| 誰かのojtチャンネルに反応する | 他の人のojtチャンネルの投稿にコメントやステッカーを送る |
| 雑談タイムに参加 | オンライン/オフラインの雑談タイムに1回参加する |

## ポイント設計

| 達成 | Meguri獲得 |
|---|---|
| クエスト1つ達成 | 500pt |
| カテゴリコンプリート | ボーナス 1,000pt |
| 全クエストコンプリート | ボーナス 3,000pt |

全クリで合計約21,000pt獲得できます。最初の学びへの投資原資がオンボーディングで貯まる設計です。

## Meguri月次ポイントの解禁

[Meguriの月次自動付与（2,000pt/月）](/docs/finance/meguri)は、オンボーディングクエストを一定数達成した時点で解禁されます。まずクエストを通じてTandenの学びの循環を体感してから、月次の循環に合流する流れです。

## 進捗の可視化

クエストの進捗はMeiboのプロフィールに表示されます。本人だけでなく周囲の人も見られるので、「まだマイコーチ設定してないな、声かけよう」というサポートが自然に起きます。
