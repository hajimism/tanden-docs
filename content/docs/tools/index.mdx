**Tanden Toolkit** — Toolkit for Context Management.

Tandenの自作ツール群の総称です。コンテキストマネジメントを実践するためのツールキットとして、14のプロダクトが深く統合されています。

## 設計思想

社内ツールは単に機能を提供するだけでなく、どういう組織でありたいかコンテキストを提供するものです。
たとえば日報の社内ツール Tsuzuri には、その存在自体に「言語化を訓練してね」「他人の学習を促してね」「積極的に自己開示してね」と伝える役割があります。

コモディティな機能はSaaS、バリューを体現する機能は自作でデザインします。
OSS のセルフホスト+拡張も積極的に検討します。

## 深い統合

Tanden Toolkitの各プロダクトは独立したツールではなく、互いに深く統合されたひとつのシステムとして動きます。Meiboに登録された人がKodouで時間を刻み、MotodeとKodouがWarifuにコストをチャージし、OkuriとTsuzuriのステッカーがMeguriポイントに変わり、すべてがKaruteに集約され、Shiorie が全体を横断する——データが途切れることなくツール間を流れる設計です。

Tanden（丹田）を起点に、Meibo（名簿）でエネルギーが流れ、Meguri（巡り）で循環し、Warifu（割符）で予算を割って委ね、Kodou（鼓動）で時間の投資先を刻む——身体とお金のメタファーがTanden Toolkitを貫いています。
