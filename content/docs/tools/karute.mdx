---
title: Karute（カルテ）
description: 会社の健康状態を可視化する全社ダッシュボード
---

**組織カルテ。** 会社の健康状態を可視化する全社ダッシュボード。財務だけでなく、関係性・学び・自律の循環まで映し出す。

## 表示項目アイデア

### 財務

| 指標 | データ取得元 |
|---|---|
| 売上 | 会計SaaS API |
| 利益率 | 会計SaaS API |
| 生産性（粗利÷社員数） | 会計SaaS + Keiraku |

### 組織

| 指標 | データ取得元 |
|---|---|
| 社員数（推移） | Keiraku |
| 入社数・退社数（月次） | Keiraku |
| 平均在籍期間 | Keiraku |

### 関係性

| 指標 | データ取得元 |
|---|---|
| Okuriの総数（月次） | Okuri |
| 一人あたりOkuri送受信数 | Okuri + Keiraku |
| Okuri未送信者率（孤立の兆候） | Okuri + Keiraku |
| マイコーチ設定率 | Keiraku Connections |

### 学び

| 指標 | データ取得元 |
|---|---|
| ステッカー流通量（月次） | Meguri |
| Meguriポイント消費率（貯め込まず使えているか） | Meguri |
| Tsudzuri外部公開数 | Tsudzuri |
| 書籍・カンファレンス投資総額 | Motode + Meguri |

### 自律

| 指標 | データ取得元 |
|---|---|
| テイアン起票数（月次） | Teian |
| テイアン平均レビュー参加人数（巻き込み力） | Teian |
| Noroshi消化率（掲載から1ヶ月以内に人が決まった割合） | Noroshi |
| Motode事後レビュー指摘率（低いほど自律的判断の精度が高い） | Motode |

### 日常

| 指標 | データ取得元 |
|---|---|
| Tsudzuri投稿率（毎日書いている人の割合） | Tsudzuri |
| Tsudzuri空欄率（高すぎると力が枯れている兆候） | Tsudzuri |
| 気分絵文字の分布（組織全体のムードマップ） | Tsudzuri |
| 対面全社会の参加率 | カレンダー / 手動 |
