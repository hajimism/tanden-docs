## 概要

**組織カルテ。** 会社の健康状態を可視化する全社ダッシュボードです。財務だけでなく、関係性・学び・自律の循環まで映し出します。

このツールが伝えるコンテキストは**「会社は今こういう状態だ」**です。管理者向けのレポートではなく、全社員が[コンテキストを受け取る](/docs/culture/values/context)ための数字です。各自が自律的に判断するための材料を届けます。

## 表示項目アイデア

### 財務

| 指標 | データ取得元 |
|---|---|
| 売上 | 会計SaaS API |
| 利益率 | 会計SaaS API |
| 生産性（粗利÷社員数） | 会計SaaS + Meibo |

### 組織

| 指標 | データ取得元 |
|---|---|
| 社員数（推移） | Meibo |
| 入社数・退社数（月次） | Meibo |
| 平均在籍期間 | Meibo |

### 関係性

| 指標 | データ取得元 |
|---|---|
| Okuriの総数（月次） | Okuri |
| 一人あたりOkuri送受信数 | Okuri + Meibo |
| Okuri未送信者率（孤立の兆候） | Okuri + Meibo |
| マイコーチ設定率 | Meibo Connections |

### 学び

| 指標 | データ取得元 |
|---|---|
| ステッカー流通量（月次） | Meguri |
| Meguriポイント消費率（貯め込まず使えているか） | Meguri |
| Tsuzuri外部公開数 | Tsuzuri |
| 書籍・カンファレンス投資総額 | Motode + Meguri |

### 自律

| 指標 | データ取得元 |
|---|---|
| テイアン起票数（月次） | Teian |
| テイアン平均レビュー参加人数（巻き込み力） | Teian |
| Noroshi消化率（掲載から1ヶ月以内に人が決まった割合） | Noroshi |
| Motode事後レビュー指摘率（低いほど自律的判断の精度が高い） | Motode |

### 日常

| 指標 | データ取得元 |
|---|---|
| Tsuzuri投稿率（毎日書いている人の割合） | Tsuzuri |
| Tsuzuri空欄率（高すぎると力が枯れている兆候） | Tsuzuri |
| 気分絵文字の分布（組織全体のムードマップ） | Tsuzuri |
| 対面全社会の参加率 | カレンダー / 手動 |

## シオリエ連携

[シオリエ](/docs/tools/shiorie)に話しかけるとKaruteの指標を引き出せます。

- 「今月の売上は？」→ 財務指標を参照
- 「組織の健康状態教えて」→ 各カテゴリの指標をサマリー
