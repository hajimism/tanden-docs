## 概要

**チャット＆コミュニケーション。** [Tanden Toolkit](/docs/tools)の一部として、リアルタイムメッセージングと全ツールの操作を一体化したコミュニケーション基盤です。[Shiorie](/docs/tools/shiorie)がネイティブ統合されており、会話の流れの中からToolkitの全機能を呼び出せます。

このツールが伝えるコンテキストは**「一人の声が組織のうねりになる」**です。日常の会話・共有・議論がそのまま組織を動かす力になります。チャットは情報を流すパイプではなく、うねりを生む場です。

## Slackとの違い

Uneriは汎用チャットツールではなく、Tanden Toolkitのために設計された専用チャットです。

| | Slack + ボット | Uneri |
|---|---|---|
| Shiorie | 外付けボット | ネイティブ統合。どのチャンネルにも常駐 |
| ツール操作 | ボットコマンド経由 | 会話の文脈からシームレスに起動 |
| 通知 | Webhook連携 | Toolkit内の全イベントがネイティブ通知 |
| ojtチャンネル | 手動作成 | 入社時に自動生成 |
| チャンネル設計 | 自由（カオスになりがち） | Toolkit連動の構造化チャンネル |

## 要件

### コアメッセージング

| 機能 | 説明 |
|---|---|
| チャンネル | パブリック / プライベート |
| DM | 1対1 / グループDM |
| スレッド | メッセージへの返信スレッド |
| リッチテキスト | マークダウン、コードブロック、引用 |
| リアクション | 絵文字リアクション（カスタム絵文字対応） |
| ファイル共有 | 画像・ドキュメント・動画のアップロード |
| メンション | @個人、@グループ、@channel、@here |
| メッセージ編集・削除 | 送信後の編集と削除 |
| ピン留め | チャンネル内の重要メッセージを固定 |
| ブックマーク | 個人用の保存メッセージ |
| 検索 | メッセージ・ファイルの全文検索、フィルタ付き |
| リンクプレビュー | URL貼り付け時のOGP表示 |

### リアルタイム通話

| 機能 | 説明 |
|---|---|
| ハドル | チャンネル/DM内の軽量音声通話 |
| ビデオ通話 | カメラON対応 |
| 画面共有 | ハドル/ビデオ通話中の画面共有 |
| 自動文字起こし | 通話内容をリアルタイムで文字起こし。要約は `@shiorie` に依頼 |
| ハドルメモ | 通話中の共有メモ |

### チャンネル管理

| 機能 | 説明 |
|---|---|
| チャンネルカテゴリ | セクション分け（プロジェクト、レーベル、雑談など） |
| チャンネルアーカイブ | 不要になったチャンネルを非表示 |
| チャンネルトピック / 説明 | チャンネルの目的を明記 |
| デフォルトチャンネル | 入社時に自動参加するチャンネル |
| チャンネルブックマーク | チャンネル上部にリンクをピン |

### ユーザーグループ

[Meibo](/docs/tools/meibo)のGroupsがユーザーグループとして機能します。Meiboでグループを作成・編集すると、Uneri側のメンショングループに自動反映されます。Uneri独自のグループ管理は持ちません。

### ステータス・プロフィール

| 機能 | 説明 |
|---|---|
| ステータス | 絵文字+テキストで自由設定（「集中中」「出張中」など） |
| Do Not Disturb | 通知一時停止（時間帯設定可能） |
| プロフィール | Meiboのプロフィールをチャット内で参照可能 |

### 生産性

| 機能 | 説明 |
|---|---|
| リマインダー | 自分へのリマインダー設定（時間指定） |
| メッセージリマインダー | 特定メッセージを後で通知 |
| 予約送信 | 指定時刻にメッセージ送信 |
| 未読管理 | チャンネルごとの未読バッジ、全既読 |

### Shiorie ネイティブ機能

| 機能 | 説明 |
|---|---|
| @shiorie 呼び出し | どのチャンネルでもメンションで起動 |
| 文脈認識 | 会話の流れを読んで適切なToolkit操作を提案 |
| Toolkit操作の実行 | チャットからTeian起票、Kodou入力、Motode精算等 |
| スレッド内応答 | 会話を遮らずスレッドで返答 |
| フィードバック | 👎リアクションで回答品質を改善 |

### Toolkit連動

| 機能 | 説明 |
|---|---|
| ojtチャンネル自動生成 | Meibo入社時に作成。Tsuzuri・Okuriの連携先 |
| テイアンスレッド自動生成 | Teian起票時にチャンネルへスレッド作成 |
| Warifuチャンネル手動作成 | Warifu上からチャンネルを作成可能（自動生成はしない） |
| Tsuzuri連携 | 日報投稿をojtチャンネルに自動転記 |
| Okuri配信 | オープンOkuriをojtチャンネルに配信 |
| Noroshi通知 | 新規JD掲載時にチャンネルへ自動投稿 |
| Karute アラート | 異常値検出をチャンネルへ通知 |
| Soubi完了通知 | シート付与完了をDMで通知 |
| 全ツール通知 | Toolkit内の全イベントを集約。通知粒度は個人ごとにカスタマイズ可能 |

### 外部連携

| 機能 | 説明 |
|---|---|
| ゲストアクセス | 特定チャンネルに外部ユーザーを招待 |
| Webhook受信 | 外部サービスからの通知受け取り |

### 管理・セキュリティ

| 機能 | 説明 |
|---|---|
| Meibo SSO | Meiboアカウントでログイン |
| 監査ログ | 誰がいつ何をしたかの記録 |
| データエクスポート | メッセージ・ファイルの一括エクスポート |

### クライアント

| 機能 | 説明 |
|---|---|
| Webアプリ | ブラウザで利用（メインクライアント） |
| PWA | モバイル・デスクトップ向けPWA対応 |

## チャットからの操作例

```
あなた: 明日から大阪出張なんだけど
Shiorie: 出張テイアンを起票しますか？
        チャージ先のWarifuコードと概算予算を教えてください。
あなた: PJT-ACMQ1、交通費と宿泊で3万くらい
Shiorie: 出張テイアンを起票しました。
        レビュアーにPJT-ACMQ1の責任者（田中さん）を指名しています。
```

```
あなた: 今日の鼓動入れるの忘れてた
Shiorie: 今日の出退勤とチャージ先を教えてください。
あなた: 9時-18時、PJT-ACMQ1に6h、SLFに2h
Shiorie: Kodouに記録しました。Tsuzuriもまだ未提出ですが、一緒に書きますか？
```

## シオリエ連携

Uneri自体がShiorieのホームです。他ツールのシオリエ連携で記載されている操作はすべてUneriから実行できます。
