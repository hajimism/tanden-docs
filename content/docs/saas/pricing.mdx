## 料金の考え方

Tanden Toolkitは**統合されていること自体が価値**です。個々のツール機能ではなく、14のプロダクトがひとつのシステムとして動く体験に対して価格をつけています。

- 全ツールで機能制限なし（プランによる出し分けはしません）
- [Meibo](/docs/tools/meibo)は認証・ID基盤として全顧客必須です
- [Shiorie](/docs/tools/shiorie)は各ツールに内包されています。導入ツールが増えるほどShiorieの対応範囲が広がり、体験が質的に変わります

## フルスイート

**¥5,500/人/月**（年払い **¥4,950/人/月**）

全14ツール + Meibo。Tanden Toolkitの統合効果を最大限に活かすプランです。Open Handbookに共感いただいた企業には、まずフルスイートをおすすめしています。

### フルスイートでしか得られない体験

- **Shiorieが全ツールを横断する** — 「今月の売上は？」→ Karute、「この経費どのコード？」→ Motode + Warifu、「〇〇さんの得意分野は？」→ Meibo。一つのAIがすべてのコンテキストを持ちます
- **データが途切れなく流れる** — Kodouの時間 → Warifuのコスト → Karuteのダッシュボード → Teianの意思決定。手作業の連携が一切不要です
- **カルチャーが回る** — ojtチャンネル・Okuri・Tsuzuri・Meguriが一つの循環として機能し、コンテキストマネジメントが組織に実装されます

### 個別SaaSとの比較

| 構成 | 市場相場 | Tanden Toolkit |
|---|---|---|
| HR + 勤怠 + 経費精算 + フィードバック（4領域） | ¥2,000〜3,000/人/月 | フルスイートに含まれます |
| 上記 + ワークフロー + 日報 + PJ管理 + チャット（8領域） | ¥4,500〜7,000/人/月 | フルスイートに含まれます |
| 上記 + BI + IT資産管理 + ピアボーナス + 社内公募（12領域） | 個別導入では非現実的 | **¥5,500/人/月** |

バラバラのSaaSを組み合わせても、ツール間のデータ連携は手作業になります。Tanden Toolkitは14ツールが最初から統合されている——この体験は個別導入では再現できません。
