## 料金の考え方

Tanden Toolkitは**ツール単位のシート課金**です。必要なツールだけ導入でき、導入ツール数が増えるほどディスカウントが適用されます。

- 全ツールで機能制限なし（プランによる出し分けはしません）
- [Meibo](/docs/tools/meibo)は認証・ID基盤として全顧客必須です
- [Shiorie](/docs/tools/shiorie)は各ツールに内包されています。単体売りはしません。導入ツールが増えるほどシオリエの対応範囲が広がります

### 必須依存について

一部のツールは単体では機能せず、他のツールの導入が前提になります。

- **Kodou → Warifu**：時間をWarifuコードにチャージするため
- **Motode → Warifu**：経費をWarifuコードにチャージするため
- **Meguri → Okuri or Tsuzuri**：ポイントの発生源（Okuri送信 or ステッカー）が必要なため

必須依存以外にも、ツール間の統合によって体験が向上する組み合わせがあります。推奨セットを参考にしてください。

| 1〜3 | 定価 |
| 4〜6 | 10%OFF |
| 7〜9 | 15%OFF |
| 10〜12 | 20%OFF |

## 市場比較

バラバラのSaaSを導入した場合との比較です。

| 組み合わせ例 | 市場相場合計 | Tanden Toolkit |
|---|---|---|
| HR + 勤怠 + 経費精算 + フィードバック（4ツール） | ¥1,800〜2,400/人/月 | ¥1,700/人/月（4ツール10%OFF） |
| 上記 + ワークフロー + 日報 + プロジェクト管理（7ツール） | ¥3,500〜5,000/人/月 | ¥2,785/人/月（7ツール15%OFF） |
| フルスイート | 個別導入では実現不可能 | ¥3,300/人/月 |

個別SaaSと同等以下の価格で、深く統合されたツール群が手に入ります。統合されていること自体がプレミアムです。
