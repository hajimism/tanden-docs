## 現在のフェーズ

現在は**Phase 0：思想の言語化**です。

| Phase | 内容 | 状態 |
|---|---|---|
| **0：思想の言語化** | オープンハンドブックの整備、経営思想の明文化 | 進行中 |
| **1：社内ドッグフーディング** | Tanden Toolkitを社内で本格運用し、磨き込む | 未着手 |
| **2：パイロット導入** | 共感してくれた数社にツール＋組織コンサルを提供 | 未着手 |
| **3：正式ローンチ** | 料金体系の確定、セールス・CS体制の構築 | 未着手 |

### なぜハンドブックが先か

CDレーベルが届けるのは「コンテキストマネジメント」という経営スタイルです。ツールはそのスタイルを実装する手段にすぎません。まず思想を明文化し、自分たちで実践し、その過程で「何がコアで、何が各組織に合わせてカスタマイズされるべきか」の境界線を見極めます。

この順番を逆にすると、抽象化の場所を間違えます。

## Phase 0 でやっていること

- オープンハンドブックの執筆・公開
- [Tanden Toolkit](/docs/tools) の設計仕様の策定
- [コンテンツ戦略](/docs/saas/marketing)に基づく情報発信の仕組みづくり
- [料金構想](/docs/saas/pricing)の検討

## 目指している方向

- 社内で使い倒す中で磨かれたツールを、他の組織でも使える形にしていきます
- 各組織の文化に合わせてカスタマイズできる柔軟な設計を志向しています
- OpenHandbookを読んで共感した組織が、自然と導入に至るような流れをつくります
